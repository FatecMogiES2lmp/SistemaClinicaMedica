Título: Sistema para clínica médica

Descrição: Sistema de gerenciamento de dados para das consultas de uma clínica médica. O sistema tem como objetivo melhorar o controle de agendamentos das consultas, facilitar e agilizar a emissão de receitas e laudos médicos, 
	facilitar o acesso ao histórico de receitas e laudos médicos repassados aos clientes da clínica através do armazenamento do banco de dados.
